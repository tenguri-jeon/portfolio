# PORTFOLIO

react를 사용하여 portfolio 페이지를 구현 해 보았습니다.

## 개발 기간

- 2024-10-17 ~ -2024-11-2 (프로젝트가 업데이트 됨에 따라 계속 수정중입니다.)

## 개발 환경
- node v20.11.1
- react-redux v 18.3.1
- sass v1.80.4
- vite v5.4.9
- framer-motion v12.0.0-alpha.0
